# Vezetői jogosultságok létrehozása és beállítása

## Jogosultsági szerepkörök

1. Egyesületvezető - ugyanaz mint az admin szerepkör (az admin legyen átnevezve egyesületvezetőnek)
2. Egyesületvezető helyettes
3. Pénzügyi vezető
4. Jogi vezető
5. Kommunikációs vezető
6. Szakszövetségi vezető

# Szerepkörök jogosultságai

## Egyesületvezető és egyesületvezető helyettes
- Teljes admin jogosultság, mindenre szerkesztési módosítási és törlési jog

## Pénzügyi vezető
- Pénzügyek - szerkesztési, létrehozási, törlési jog az admin felületen
- Tagdíjak és részvételi díjak befizetéseinek adminisztrációs joga
- Dokumentumtár szerkesztési joga, feltöltés, törlés
MINDEN MÁS: csak megtekintés

## Jogi vezető
- Dokumentumtár szerkesztési joga, feltöltés, törlés
- Részvételi feltételek szerkesztési joga
MINDEN MÁS: csak megtekintés

## Kommunikációs vezető
- Bejegyzések létrehozás, módosítása törlése (hír és élményblog)
- GYIK szerkesztése
- Lapok szerkesztése (Adó 1%, Klubélet, Rólunk, Tagság)
MINDEN MÁS: csak megtekintés

## Szakszövetségi vezető
- Túrák szerkesztése
- Beérkező túrák jóváhagyása, elutasítása
MINDEN MÁS: csak megtekintés

# Új logika a vezetőknek

- A vezető szerepkörrel rendelkező tagok jelenjenek meg a Rólunk szekcióban profilképpel, vezetőségi tisztségének megnevezésével és elérhetőségekkel
